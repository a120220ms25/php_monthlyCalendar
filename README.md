# Perpetual-calendar

## [Demo](http://220.128.133.15/s1100218/backEnd/php_calender/?year=2021&month=07)

## 簡介
以PHP撰寫的萬年曆系統，可以點取連結切換不同月份，以及點選連結回到現在時間之月曆。 並利用css、BS5來美化畫面並且做到rwd響應式效果。

## 主要使用技術
* PHP
